# Boltzmann Wealth Model

Boltzmann Wealth modeled using [Mesa](https://mesa.readthedocs.io/en/stable/index.html) agent-based modeling library,
based on [this](https://mesa.readthedocs.io/en/stable/tutorials/intro_tutorial.html) tutorial.

## Setup

1. Create virtual environment: `python -m venv .venv`
2. Activate virtual environment: `source .venv/bin/activate`
3. Install requirements: `pip install -r requirements.txt`
